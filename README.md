# Overview

Sudarshan Nano AI (SDN-AI) is a lightweight workflow inference engine designed for real-time operational automation.

Unlike large conversational AI systems that require significant GPU infrastructure, SDN-AI focuses on:

- Ultra-fast workflow inference
- Lightweight semantic processing
- Intent detection
- Ticket routing
- Real-time operational automation
- CPU-first deployment
- Extremely low memory usage

The project explores practical AI infrastructure for operational systems where:

- latency matters
- infrastructure cost matters
- deployment simplicity matters
- real-time execution matters
