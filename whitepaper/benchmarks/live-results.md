# Live Benchmark Results — Sudarshan Nano AI

## Overview

This document contains observed operational metrics from the live public demonstration of Sudarshan Nano AI (SDN-AI).

Live Demo:

https://sdnai.saiinfosoft.co.in/

---

# Benchmark Philosophy

The purpose of these observations is NOT to compare conversational intelligence capability with large language models.

Instead, these metrics demonstrate:

- lightweight operational inference
- CPU-first deployment
- low memory workflow execution
- real-time automation behavior
- practical workflow latency characteristics

---

| Metric | Observation |
|---|---|
| Average Latency | ~0.48–0.66 ms |
| Peak Memory Usage | ~63–73 KB |
